Combinador de horarios
Consideraciones:
  1. El uso de este sistema es para generar horarios.
  2. El presente sistema tiene por finalidad: usar filtros, generar horarios segun filtros, diseña los horarios por semana y lista los horarios.
  3. Los datos que utiliza este sistema es unicamente del archivo Excel.
  4. No se acaptan atribuciones de responsabilidad externa al sistema.
  5. Ultima actualizazion: primer semestre de 2021
Requisitos:
  1. Archivo con extensión .xlsx o .xls
  2. Leer cada parte de la presente pagína web.
  3. Tener en una hoja Excel con una fila de los nombres de cada columna, y desde de esa fila tener los datos correspondientes.
  4. Los espacios de los nombres de las columnas, cambiarlos por un texto reconosible.
  5. Los contenidos que esten en blanco, cambiarlos por un texto reconosible, verifique el contenido de la columna Nivel.
